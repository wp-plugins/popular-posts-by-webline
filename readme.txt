Popular Posts will display the post according to the filters applied from widget setting. 

== Description ==

Popular Posts will list all the post as per filters applied from the admin side.

= Key Features =

* It display post by applying the filters available like Sort By comments posted on post.
* It also has filter for date which shows post as the time span(In Days) selected
* Post can also be displayed as per the category selection

== Installation ==

* Upload the Display Plugin and activate it from the Plugin option available in backend.
* You can also unzip and place it in the plugin directory.
